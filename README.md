Верстка проекта интернет-магазина косметики.
В дальнейшем данная верстка была натянута на CMS OpenCart для полноценного функционирования сайта.

Были использованы:

- HTML, CSS, JS, JQuery;
- Bootstrap;
- Fancybox;
- Slick;
- API Яндекс карт.
